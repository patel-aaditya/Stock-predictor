# Instructions (Spyder)

1. Open **Spyder**.
2. Open the file: `main.py`.
3. Run the file (green play button).
4. Enter a ticker symbol (e.g., `AAPL`).
5. Charts will pop up in separate windows.

## Troubleshooting
- If you see `Missing dependency: yfinance`, install:
  ```powershell
  pip install yfinance
  ```
- If charts donâ€™t appear, ensure `matplotlib` is installed:
  ```powershell
  pip install matplotlib
  ```
- If no data returns, check the ticker symbol.
